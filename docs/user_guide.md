
# Guía de Usuario - NeoGenesis

Bienvenido al sistema NeoGenesis. Esta guía te ayudará a navegar por la interfaz y sacar el máximo provecho a tus generaciones.

## 1. Selección de Modo
Al iniciar, verás 3 interruptores principales en la parte superior:
*   **QUICK vs ADVANCED:** Elige entre un asistente paso a paso o el panel de control completo.
*   **IMAGE vs VIDEO:** Cambia la estructura del prompt. "Video" prioriza acciones continuas ("Walking", "Fighting") sobre poses estáticas.
*   **MIDJOURNEY vs GENERIC:**
    *   *Midjourney:* Añade parámetros técnicos como `--ar 16:9 --v 6.0`.
    *   *Generic:* Usa lenguaje natural ("Cinematic lighting, 8k") compatible con DALL-E 3 o Stable Diffusion.

## 2. Modo Avanzado (Advanced Grid)
El panel se divide en 5 columnas lógicas:

### Columna 1: Core Identity
Define quién es tu personaje.
*   **Raza & Género:** Base biológica.
*   **Clase (Fantasy vs Realistic):** El interruptor cambia las listas de roles. Si eliges "Fantasy", verás Magos y Guerreros. Si eliges "Realistic", verás Soldados y Doctores.

### Columna 2: Face & Head
Detalles del rostro.
*   **Colores Simétricos:** Ahora puedes elegir hasta **2 colores** para Piel, Pelo y Ojos. Esto permite crear personajes con heterocromía (un ojo de cada color) o piel con patrones alienígenas.

### Columna 3: Outfit & Gear
Vestimenta y equipamiento.
*   **Outfit Colors:** Selector global para la paleta de la ropa.
*   **Class Extras:** Accesorios específicos como Alas, Mochilas o Auras.

### Columna 4: Style & Vibe
La dirección artística.
*   **Estilo:** Desde "Dark Fantasy" (Elden Ring) hasta "Pixar 3D".
*   **Color Palette:** Define la iluminación y el tono general de la imagen (no del personaje, sino de la escena).

### Columna 5: World & Render
El escenario.
*   **Setting:** Dónde está el personaje.
*   **Aspect Ratio:** Formato de la imagen (16:9 cine, 9:16 móvil, etc.).

## 3. Botones de Generación

*   **GENERATE IMAGE:** Crea un único prompt detallado y optimizado.
*   **GENERATE INVENTORY:** Crea un prompt para una imagen de tipo "Knolling" (objetos organizados sobre fondo blanco) con el equipo de tu personaje.
*   **PSYCHE PROTOCOL (SHEETS):** La joya de la corona. Genera 7 prompts distintos para crear una hoja de personaje completa:
    1.  Vista Arquitectónica (Frente/Lado/Espalda).
    2.  Token VTT (Para juegos de rol).
    3.  Expresiones faciales.
    4.  Poses de acción.

## 4. Herramientas Útiles
*   **CRT_FX:** Botón superior derecho. Activa/Desactiva el efecto de monitor retro.
*   **MEMORY:** Abre el historial lateral para recuperar prompts anteriores.
*   **ELITE AGENT:** Carga un personaje pre-diseñado de alta calidad al azar.
*   **GENOME CHAOS:** Aleatoriza todos los parámetros para inspirarte.
